# REQ-002: Author Mapping

## Description
Map CVS usernames to Git author information (name and email).

## Acceptance Criteria
- [ ] Can load author mapping from config
- [ ] CVS usernames are mapped to Git authors
- [ ] Unmapped authors use default format
- [ ] Can extract authors from CVS repository
- [ ] Can generate mapping template

## Test Cases
1. Load author mapping from config
2. Map CVS user to Git author
3. Handle unmapped users
4. Extract authors from repository
5. Generate mapping template
